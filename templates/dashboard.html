Project Overview

EduTrack is a Smart Attendance and Internal Performance Monitoring System. It stores student information in a database and automatically calculates Attendance Percentage, Internal Percentage and Academic Attendance Index (AAI). Faculty can monitor student performance using a single Roll Number.
